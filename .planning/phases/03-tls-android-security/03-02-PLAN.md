<objective>
Encrypt the auth token at rest on Android using Android Keystore + AES-256-GCM, replacing plaintext DataStore storage.

Purpose: Prevents token extraction via filesystem inspection (rooted device, adb shell, backup extraction). The token is the sole authentication gate -- its compromise grants full shell access to the host machine.

Output: TokenEncryptor utility class, updated SettingsStore with encrypted token flow, unit tests with mock encryption layer.
</objective>
